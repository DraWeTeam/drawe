# 0004 — latent→geometric observable: 모호한 관찰축을 기하 binary로 재정의해 측정 정의 붕괴를 막는다

상태: 채택 (2026-06-30)

## 맥락

VLM 관찰자([0001])의 관찰 *필드*가 자유서술이면, 거친 축이 일치해도 *무엇을 봤는지*가 호출마다
달라질 수 있다. `observe_hand`의 단축(foreshortening)이 그랬다. 처음엔 `foreshortening`을 **자유
리스트**("단축돼 보이는 손가락/부위", 예: `["중지","약지"]`)로 받았는데:

- **presence는 안정**(단축 케이스 4/4에서 "뭔가 단축됨"은 일관) — 모델 한계가 아니다.
- 그런데 *어느 손가락*이 단축이라 지목하는지는 호출마다 흔들렸다. 단축은 **latent concept**(모호한
  잠재 속성)이라 VLM이 매 호출 다른 항목을 보고한다 = **측정 정의가 붕괴**(비교·집계할 고정 대상이 없음).
- [0001]의 2-run 교집합 게이트는 이 노이즈를 못 눌렀다 — 두 run의 손가락 집합이 어긋나면 교집합이
  비어 surface가 1/5~4/5로 흔들렸고, 시각상 최강 케이스가 오히려 억제됐다.
- 게다가 단축 판단이 **view**(손등/손바닥/옆면)에 묶여(view-coupling), view가 흔들리면 단축까지 같이
  떨어졌다 — 단축은 깊이(원근) 관찰이라 손등인지 손바닥인지 알 필요가 없는데도.

붕괴의 위치가 [0001]·[0003]과 다르다: [0001]은 측정 vs 관찰의 *경로* 분리, [0003]은 over-fire/abstain의
*게이팅*. 여기서 깨진 건 관찰자 패턴 *안에서* 관찰 **필드를 어떻게 정의하면 붕괴를 막나**다.

## 결정

**자유서술 latent 속성을, 셀 수 있는 기하 관찰값(geometric binary observable) 소수로 재정의한다.**
단축의 경우(`ml/vision.py` `observe_hand` 실제 필드):

- `reaching_at_viewer`(예/아니오/불확실) — 손·손가락이 보는 사람 쪽으로 뻗어 깊이 축이 있나.
- `parts_compressed`(예/아니오/불확실) — 원근으로 길이가 짧게 눌리거나 겹쳐 보이나.
- 둘을 **3단 tier**로 합성(`_fs_tier`): 둘 다 예=`STRONG` / 하나만=`WEAK` / 아니면=`NONE`. threshold-free.
- 두 run의 tier는 **보수 합의**(min rank)로 grounding — [0001]의 신뢰 3단을 *기하 관찰* 위에 다시 건다.
- 자유 리스트 `foreshortening`은 **디버그/표기용으로만 보존**(게이트에서 미사용).

그리고 **교란변수(view)를 디커플**한다. `diagnose.py` `_vlm_hand_signal`에서 view/structure surface는
view 신뢰(`view_ok`)가 담당하고, 단축 surface는 `foreshortening_tier == "STRONG"`이 view와 **독립**으로
담당한다(view가 '낮음'이어도 STRONG이면 발화). 단정 없이 tier로만 — 부위는 안 짚는다.

이건 단축에 국한된 fix가 아니라 **재사용 템플릿**이다: *관찰축이 자유서술 출력으로 정의 붕괴(매 호출
다른 항목 보고)할 때 → 소수의 기하 binary로 재정의 + 교란변수 디커플*. [0001]의 "자유 텍스트는 비교
불가 → 구조화"를 한 단계 더 밀어, 구조화된 필드조차 latent면 **기하 관찰로 내린다**.

## 기각한 대안

- **자유서술 유지 + 후처리 정규화.** 기각: 매 호출 항목이 바뀌어 **정규화할 고정 대상 자체가 없다**.
  교집합·정규화는 같은 것을 두 번 봤다는 전제인데 그 전제가 깨진 게 문제였다.
- **CV로 단축을 직접 측정(기하 비율).** 기각: 골든셋 테스트베드 실측에서 **기하 비율만으론 단축을 분리할
  수 없음**이 드러났다 — "짧은 손가락"과 "단축된 손가락"이 둘 다 손가락/손바닥 비율↓로 나타나 기하학적
  구분이 안 된다(긴/짧은 finger-proportion을 CV로 닫는 과정에서 확인 — 단축 자체와는 다른 결함이지만,
  *기하 비율 단독으론 단축이 안 갈린다*의 근거). 그래서 측정은 CV가 아니라 VLM의 기하 관찰로 둔다.
- **view까지 한 신호에 묶기(현행 유지).** 기각: 단축이 view 흔들림에 끌려 떨어지는 view-coupling이 곧
  surface 불안정의 절반이었다. 깊이 관찰과 평면 방향은 직교라 분리해야 한다.

## 결과

- (+) 단축 surface가 N=5 테스트베드에서 1/5~4/5 → **5/5** 안정, over-fire 0(정상 손 0/5). "정의를 고정하면
  VLM이 일관"이 실증됨 — 흔들린 건 모델이 아니라 정의였다.
- (+) **재사용 템플릿 확보** — 다른 모호 관찰축(hand view·face eye_line 등 "자유 관측→기하 grounding" 계열)에
  같은 방법을 적용할 수 있다.
- (−) 기하 binary로 좁히면 **잡는 결함이 제한**된다: 긴/짧은 같은 continuous 속성은 categorical binary로
  못 잡는다(VLM 3분류는 경계가 정상값에 걸쳐 over-fire). 그래서 finger-proportion은 측정 후 정직히 닫았다
  — latent→geometric은 *이산 기하 관찰로 환원되는* 축에서만 작동한다.
