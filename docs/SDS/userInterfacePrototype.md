## 5.2 인터랙션 포인트
- **레퍼런스 그리드**: 검색 결과 `[N]` 번호 ↔ 응답 본문 인용 일치. 핀 시 "고정 N" 슬롯.
- **새 프로젝트 키워드 칩**: AI가 제안한 키워드를 사용자가 X로 제거하거나 직접 추가.
- **검색 모달**: 입력 즉시 결과, 1회 이상 검색 시 최근 검색어 노출.
- **가이드 툴팁**: 진단 축(구도·명암·색감 등)에 대한 설명 오버레이.
