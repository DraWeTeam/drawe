**★핵심 5건 요약**
- **#4 키워드 3개·대그룹 오렌지**: ⑥ track_map(group) 확보로 즉시 구현 가능.
- **#5·#9 개선 포인트 2개 리스트**: 백엔드가 2개 개선포인트/2~3 체크항목을 구조화해 내려야 함(현재 effect 1문장·focus_practice 1개).
- **#8 이론 질문 조건부 렌더 (guideAsset SVG 실측)**: `blocks.guide_asset`은 **백엔드에서 옴**(primary.guide_asset → AssetSvg 렌더 확인). 미구현은 (a) 사용자 이미지 위 오버레이 + (a)/(b) 분기. OverlayImage 컴포넌트는 import만 되고 미렌더.
- **#19 모아보기 필터탭**: 대그룹 필터/태그 미구현(코드 주석에 명시적 보류). ⑥로 해소 가능.
- **#11 담기→보드 아카이브 필터 연동**: addReference는 호출되나 보드 '아카이브' 필터 노출은 실렌더 확인 필요(토큰 대기).

## 2. 타 화면 미반영 갭 목록 (annotation 기준, 구현 미검증 — 목록화)

### SCR-GUIDE-02 가이드 결과 카드 ★
- 가이드 모아보기 아이콘: 가이드 1회 이상 생성 시 노출 [상태]
- 가이드 카드: 채팅 인라인 요약 카드, 제목·본문 …초과 표시, 썸네일=업로드 그림(비율 유지 크롭) [상태]
- 카드 클릭 → SCR-GUIDE-03 / 이미지 클릭 → 02-1 [이동]
- **PDF 다운로드 버튼**(가이드를 PDF로) [동작] — 전체화면 카드에만 annotation
- 가이드 모아보기 툴팁: 카드 1회 발행 시 최초, X로만 소멸 [동작]
- scroll to bottom 버튼 [동작]

### SCR-GUIDE-02-2 가이드 결과 모아보기(우측 패널) ★

### SCR-AUTH-01~04 / SCR-JOIN-01~04 ☆(auth 영역)
